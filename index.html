מתמחים טופ AI

הסקריפט האולטימטיבי לסיכום אשכולות ארוכים באמצעות בינה מלאכותית (Gemini Flash). מהיר, חכם ומשתלב בשלמות בעיצוב הפורום.

התקן עכשיו (גרסה 9.1)
דורש תוסף Tampermonkey מותקן בדפדפן

מה חדש בגרסה 9.1?

  • שאלות ותשובות (Chat): אפשרות לשאול את ה-AI שאלות על בסיס הדיון בחלון הסיכום.
  • היסטוריית גרסאות מתקדמת: דפדוף אחורה וקדימה בין סיכומים שונים שעשיתם לאותו אשכול.
  • כפתור פרסום: העברה אוטומטית של הסיכום המלא לחלון התגובות בפורום בלחיצת כפתור.
  • מצב לילה (Dark Mode): תמיכה מושלמת ודינמית בערכת הנושא השחורה של הפורום.
  • כפתור מרחף: כפתור סיכום נוסף שמופיע למטה רק כשגוללים, לנוחות מקסימלית.
  • עיצוב מחדש של שורת הכפתורים למראה נקי ומסודר יותר.

תכונות עיקריות

  • סיכום מקיף: קורא את כל עמודי האשכול (ללא הגבלת עמודים) לקבלת תמונה מלאה.
  • ממשק טבעי: שימוש ברכיבי הממשק המקוריים של NodeBB למראה שמרגיש חלק מהאתר.
  • התאמת הנחיה: אפשרות לערוך את ה-Prompt כדי לקבל סיכומים בדיוק בסגנון שלכם.
  • גלילה חופשית: סיכום ברקע שאינו חוסם את המסך עד שהוא מוכן.
  • היסטוריה גלובלית: גישה מהירה ל-20 האשכולות האחרונים שסיכמתם.

איך מתחילים?

שלב 1: התקנת הסקריפט

ודאו שיש לכם את תוסף Tampermonkey בדפדפן. לאחר מכן, לחצו על כפתור ההתקנה הירוק בראש עמוד זה ואשרו את ההתקנה בחלונית שתקפוץ.

שלב 2: הוצאת מפתח API (בחינם)

הסקריפט משתמש במודל המתקדם של גוגל. כדי להפעיל אותו, יש להוציא מפתח אישי:

  1. היכנסו לאתר Google AI Studio.
  2. לחצו על Get API key בתפריט הצדדי.
  3. לחצו על Create API key והעתיקו את רצף האותיות והמספרים.

שלב 3: הפעלה בפורום

היכנסו לאשכול כלשהו בפורום "מתמחים טופ". לחצו על כפתור "סכם נושא" החדש שיופיע בסרגל הכלים. בפעם הראשונה בלבד, המערכת תבקש מכם להדביק את מפתח ה-API שהעתקתם (הוא נשמר מקומית ובטוח בדפדפן שלכם).
