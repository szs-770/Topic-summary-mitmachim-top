
# 🤖 Mitmachim Top - AI Topic Summarizer

**סקריפט חכם לסיכום אשכולות בפורום "מתמחים טופ" באמצעות בינה מלאכותית (Gemini 3.1 Flash).**

התוסף מוסיף כפתור "סכם נושא" אלגנטי ישירות לשורת הכלים של הפורום, המאפשר לקבל סיכום תמציתי ומדויק של כל הדיון באלפיות שנייה, ללא הגבלת כמות פוסטים.

## ✨ תכונות עיקריות

* **סיכום מקיף:** סריקה של כל עמודי האשכול דרך ה-API של הפורום לקבלת תמונה מלאה.
* **ממשק טבעי (Native UI):** שימוש ברכיבי הממשק המקוריים של NodeBB (חלוניות Bootbox) למראה נקי שמשתלב בפורום.
* **מנגנון Caching חכם:** שמירה מקומית של סיכומים לטעינה מיידית בביקור חוזר.
* **פורמט ברור:** סיכום בנקודות (Bullet points) עם כותרות מודגשות לקריאה מהירה.
* **התקנה פשוטה:** עדכונים אוטומטיים דרך GitHub.

## 🚀 הוראות התקנה

1. התקן את התוסף **Tampermonkey** לדפדפן שלך.
2. לחץ על קישור ההתקנה הבא:
**[התקן את הסקריפט בלחיצה כאן](https://www.google.com/search?q=https://github.com/szs-770/mitmachim-summarizer/raw/refs/heads/main/mitmachim-ai.user.js)**
3. בחלונית שתפתח, לחץ על **Install** (התקן).

## 🔑 הגדרת מפתח API (בחינם)

הסקריפט משתמש במודל Gemini של גוגל. כדי להפעיל אותו, עליך להוציא מפתח API אישי:

1. היכנס ל-[Google AI Studio](https://aistudio.google.com/).
2. לחץ על **Get API key** ואז על **Create API key in new project**.
3. העתק את המפתח שקיבלת.
4. בפורום "מתמחים טופ", לחץ על כפתור "סכם נושא" והדבק את המפתח בחלונית שתופיע.
